# Streamlit 手工回归清单

适用范围：`app_home.py` 首页导航、`pages/2_崂应报价.py` 崂应报价页、`01-脚本文件/sheet_metal_quote_full.py` 兼容入口。

## 自动冒烟脚本

执行页面级自动冒烟：

```powershell
python -m pytest tests/test_streamlit_smoke.py -v
```

执行全量测试：

```powershell
python -m pytest
```

预期结果：
- 首页可正常加载，无异常。
- 崂应页可正常加载，关键控件存在。
- 预置一行有效数据后，点击“更新计算”可完成重算，汇总指标正常显示。

## 手工回归前准备

1. 在项目根目录执行：

```powershell
streamlit run app_home.py
```

2. 确认浏览器能打开首页。
3. 如果需要验证独立入口，再执行：

```powershell
streamlit run "01-脚本文件\\sheet_metal_quote_full.py"
```

## 首页导航验证

1. 首页应能正常显示“青岛宏泰铭润机械 · 钣金智能报价系统（多客户专属版）”标题。
2. 应能看到 4 个客户入口卡片。
3. 点击“进入崂应报价”后，应能正常进入崂应页，无白屏或 Python 报错。

预期结果：
- 首页无报错。
- 跳转到崂应页后，左侧边栏和主表区域都能正常显示。

## 崂应页正常路径

1. 在第一页或新开页面进入崂应报价页。
2. 填写一行数据：
   - `料品名称`：`页面回归件`
   - `材质`：`6061铝板`
   - `长(mm)`：`100`
   - `宽(mm)`：`100`
   - `厚度(mm)`：`2`
   - `米数(切割长度)`：`1.5`
   - `折弯次数`：`2`
   - `表面处理`：`喷粉`
   - `加工数量`：`10`
3. 点击“更新计算”。
4. 观察整单汇总区。

预期结果：
- `重量(kg)` 约为 `0.07098`
- `材料费(元)` 约为 `22.71`
- `切割费` 对应口径约为 `3.6`
- 汇总区显示：
  - 零件总数量约为 `10 件`
  - 整单总重量约为 `0.710 kg`
  - 整单最终含税总价约为 `45.83 元`

说明：
- 当前系统总价口径以现行代码为准，验证时优先核对页面计算结果与导出结果是否一致。

## 崂应页边界路径

### 必填项缺失

1. 新增一行，保留 `料品名称` 为空。
2. 点击“更新计算”。

预期结果：
- 该行序号变为红色前缀。
- 页面出现错误提示。
- 错误明细中包含“料品名称为空”或等价提示。

### 无材质列导入

1. 准备一个缺少“材质”列的 Excel 或 CSV。
2. 在“导入零件表”区域点击解析。

预期结果：
- 页面给出“未识别到材质列”提示。
- 当前报价表不应被错误覆盖。

### 极小总重系数

1. 将边栏“总重系数”调为 `0.01`。
2. 保持正常样例行不变，点击“更新计算”。

预期结果：
- `重量(kg)`、`总重(kg)`、`材料费(元)` 显著变小。
- 页面不报错。

### 零折弯、零攻丝

1. 将 `折弯次数` 改为 `0`。
2. 将 `攻丝数量` 改为 `0`。
3. 点击“更新计算”。

预期结果：
- 折弯相关费用为 `0`
- 攻丝相关费用为 `0`
- 页面总价会相应下降

## 导出验证

1. 基于正常路径样例点击“崂应表格导出”。
2. 确认 `03-导出报价文件` 下生成当天目录和导出文件。
3. 打开导出的 Excel。

预期结果：
- 文件生成成功，无“目录不可写”异常。
- 导出表头、汇总行可正常查看。
- 页面显示的总价与导出文件“合计”一致。

## 兼容入口验证

1. 分别使用以下两个命令启动：

```powershell
streamlit run app_home.py
streamlit run "01-脚本文件\\sheet_metal_quote_full.py"
```

2. 各执行一次正常路径验证。

预期结果：
- 两种入口都能正常启动。
- 崂应页核心功能一致。
