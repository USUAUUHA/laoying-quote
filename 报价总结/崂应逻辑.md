## 二、核心概念：一张报价单是怎么算出来的

### 2.1 输入字段（用户在表格里填的）

| 字段 | 说明 |
|------|------|
| 料品名称 / 料号 / 料品规格 | 标识信息，不参与计算 |
| 材质 | 决定密度和材料单价 |
| 长(mm) / 宽(mm) / 厚度(mm) | 决定单重和面积 |
| 米数(切割长度) | 激光切割长度（米） |
| 穿孔数量 / 焊接长度 / 沉孔数量 / 折弯次数 / 压铆数量 / 攻丝数量 | 各加工项数量 |
| 表面处理 | 喷粉/喷砂/氧化/抛光/镀锌/无，决定表面处理费 |
| 加工数量 |这件零件要生产多少件 |

### 2.2 输出字段（计算出来的）

| 字段 | 说明 |
|------|------|
| 重量(kg) | 单重 × 总重系数，理重 × 1.3 |
| 总重(kg) | 单重 × 加工数量 |
| 材料费(元) | 总重 × 材料单价 |
| 打磨(元) / 切割费 / 穿孔费 / 焊接费 / 沉孔费 / 折弯费 / 压铆费 / 攻丝费 | 各加工项费用 |
| 总加工费(元) | 以上所有加工费之和 |
| 表面处理费(元) | 按展开面积 × 处理单价 |
| 含税单价(元/件) | （材料费 + 加工费 + 表面处理费）÷ 数量 |
| 含税总价(元) | 含税单价 × 数量 |

## 三、计算公式

### 3.1 重量

```
单重(kg) = 长 × 宽 × 厚(mm) × 密度 ÷ 1,000,000
总重(kg) = 单重(kg) × 加工数量
```

**密度由材质决定（部分）：**

| 材质 | 密度 (g/mm³ × 10⁶) | 品类 |
|------|------|------|
| 铝板类（6061/1060/5052） | 2.73 | 铝板 |
| 不锈钢类（SUS304/SUS316 等） | 7.93 | 不锈钢 |
| 碳钢/镀锌（Q235/SGCC/冷板） | 7.85 | 碳钢/镀锌板 |

**总重系数：默认 1.3**，在侧边栏可调（与 Excel 一致）。

### 3.2 材料费

```
材料费(元) = 总重(kg) × 材料单价(元/kg)
```

材料单价由材质决定，例如：

| 材质 | 单价（元/kg） |
|------|------|
| 6061铝板 | 32.0 |
| SUS304 | 22.0 |
| Q235 / SGCC / 冷板 | 8.0 |

### 3.3 切割费

```
切割费 = 米数(切割长度) × 厚度(mm) × 切割系数(1.2) × 打磨倍率(1.13)
```
> 2026-05-22 更新：切割费结果含税（乘 1.13）

### 3.4 打磨费

```
if (长 ≤ 200 OR 宽 ≤ 200):
    打磨费 = 打磨基数小件(0.4) × 加工数量 × 打磨倍率(1.13)
else:
    打磨费 = 打磨基数大件(0.5) × 加工数量 × 打磨倍率(1.13)
```
> 2026-05-22 更新：规则改为 ≤200（包含200）；显示精度改为小数点后3位

### 3.5 穿孔费

```
穿孔费 = 穿孔数量 × 厚度(mm) × 穿孔系数(0.1)
```

### 3.6 焊接费

```
焊接费 = 焊接长度(mm) × (焊接系数A(40) + 焊接系数B(20))
```

### 3.7 沉孔费

```
沉孔费 = 沉孔数量 × 沉孔单价(0.3)
```

### 3.8 折弯费

```
if (材质是铝板):
    折弯单价 = 折弯铝板单价(0.5)
else:
    折弯单价 = 折弯钢/不锈钢单价(0.6)

if (长 > 100 OR 宽 > 100):
    折弯单价 = 折弯大件单价(0.7)

折弯费 = 折弯单价 × 折弯次数 × 加工数量
```

### 3.9 压铆费

```
压铆费 = 压铆数量 × (压铆系数A(0.3) + 压铆系数B(0.15))
```

### 3.10 攻丝费

```
if (材质是铝板):
    攻丝单价 = 攻丝铝板单价(0.4)
else:
    攻丝单价 = 攻丝钢/不锈钢单价(0.6)

攻丝费 = 攻丝单价 × 攻丝数量 × 加工数量
```

### 3.12 激光打标费

```
激光打标费 = 激光打标数量 × 打磨倍率(1.13)
```
> 2026-05-22 新增：激光打标费含税（乘 1.13），计入总加工费

### 3.13 表面处理费

展开面积：

```
展开面积(mm²) = 长 × 宽 × 2
```

按表面处理类型：

| 处理方式 | 计算公式 |
|------|------|
| 无 | 0 |
| 喷粉 | 展开面积(mm²) ÷ 1,000,000 × 25 × 数量 |
| 喷砂 | 展开面积(mm²) ÷ 10,000 × 1.2 × 数量 |
| 氧化 | 展开面积(mm²) ÷ 10,000 × 0.8 × 数量 |
| 抛光 | 展开面积(mm²) ÷ 10,000 × 4 × 数量 |
| 镀锌 | 总重(kg) × 15 |

### 3.14 最终价格

```
总加工费 = 切割费 + 打磨费 + 穿孔费 + 焊接费 + 沉孔费 + 折弯费 + 压铆费 + 攻丝费
含税总价(元) = 材料费 + 总加工费 + 表面处理费
含税单价(元/件) = 含税总价(元) ÷ 加工数量
```

## 五、数据导入

支持三种方式导入零件数据：

1. **Excel / CSV** → 通过 `data_io.py` 解析，自动识别列并填充表格
2. **图片（PNG/JPG）** → 调用 OCR 识别零件信息
3. **手动粘贴** → 用户直接粘贴数据到表格

## 六、导出

点击「崂应表格导出」生成标准崂应格式 Excel。

- 所有加工单价参数

## 七、关键设计决策

1. **更新计算是手动点击的**：用户填完表必须点「更新计算」才重算，避免每次输入都触发重算。
2. **序号错误标记**：有问题的行序号前加 🔴，不影响其他行计算。
3. **总重系数统一**：切割、材料费、所有按重量计费项目都用同一个系数，保持与 Excel 一致。
4. **加工参数保存在 `price_params.json`**：改完侧边栏单价会自动持久化，下次启动保留。
5. **多 Sheet Excel 优先读"零件总表"**：导入时自动定位，避免其他 Sheet 干扰。
6. **材质/表面处理模糊识别**：支持"铝板5052"、"喷粉"等多种写法自动匹配标准值。
